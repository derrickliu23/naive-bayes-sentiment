How it works

Naïve Bayes computes the probability of each class given the input words using Bayes' theorem, assuming each word is independent: 

log P(class | words) ∝ log P(class) + Σ log P(word | class)

Words unseen during training are handled with Laplace smoothing (α=1), which adds a small count to every word so no probability is ever exactly zero.
